The RGtk2 package works with the GTK version with the same or higher major and minor versions. For example, GTK 2.8.0 or higher is required for the RGtk2 2.8.x series.

For detailed installation instructions, please see the INSTALL file.

The RGtk2 package supports the GTK and GDK libraries, along with ATK, Pango, Cairo, and (to a lesser extent) GObject/GLib. It optionally supports libglade. The configuration for the RGtk2 package looks for pkg-config to compute the compiler and linker flags needed to create the shared libraries.
